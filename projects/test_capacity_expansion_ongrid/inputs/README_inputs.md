# Inputs folder

This folder contains user-editable input templates.

## load_demand.csv
- Hourly **load demand** template (8760 rows).
- Units: **kWh per hour** (energy during each hour).
- **Two-row header**:
  - Row 1: scenario labels
  - Row 2: year labels
- A meta column `meta/hour` provides the hour index (0..8759).

Scenarios: Constant_Demand, Moderate_Growth, High_Growth

Years: 2026, 2027, 2028, 2029, 2030, 2031, 2032, 2033, 2034, 2035, 2036, 2037, 2038, 2039, 2040, 2041, 2042, 2043, 2044, 2045

## resource_availability.csv
- Hourly **resource availability** template (8760 rows).
- Units: **capacity factor** (per unit of nominal capacity, typically 0..1).
- **Three-row header**:
  - Row 1: scenario labels
  - Row 2: year labels
  - Row 3: resource labels
- A meta column `meta/hour` provides the hour index (0..8759).

Scenarios: Constant_Demand, Moderate_Growth, High_Growth

Years: 2026, 2027, 2028, 2029, 2030, 2031, 2032, 2033, 2034, 2035, 2036, 2037, 2038, 2039, 2040, 2041, 2042, 2043, 2044, 2045

Resources: Solar

## renewables.yaml
- Renewable techno-economic parameters.
- Parameters can vary by resources, scenario and (if enabled) by investment step.

Renewables Technologies: Solar_PV


## battery.yaml
- Battery techno-economic parameters.
- Parameters can vary by scenario and (if enabled) by investment step.
## grid.yaml
- Grid connection parameters (scenario-dependent):
  - line capacity (kW)
  - transmission efficiency (-)
  - (multi-year only) first year of grid connection
  - outage statistics for backend-generated availability matrix

## grid_import_price.csv
- Hourly grid import cost (8760 rows).
- Units: currency per kWh.
- Two-row header: scenario labels / year labels.
- Meta column `meta/hour` provides hour index (0..8759).

Note: the grid availability matrix is generated by the backend from outage inputs (no user template).
